# CIFAR-10 Object Classifier

This project implements an object classifier using the CIFAR-10 dataset, which consists of 60,000 32x32 color images across 10 different classes.

## Dataset Classes
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck
